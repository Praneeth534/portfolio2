Projects

Project 1

The ExpenseTracker project is a simple Java-based application for tracking expenses. It uses Swing for the graphical user interface (GUI). The program allows users to input descriptions and amounts for their expenses, which are then displayed in a list. Users can add new expenses, and the application dynamically updates the list to show each entry. The GUI includes fields for entering a description and amount, as well as a button to add the expense to the list. This tool provides a basic way to keep track of personal spending in a visually organized manner.
